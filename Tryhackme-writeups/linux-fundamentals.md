# Room Name: Linux Fundamentals

## Objective

The objective of this room was to understand the basics of the Linux operating system, including file navigation, permissions, users, processes, and essential commands commonly used in cybersecurity.

---

## Topics Covered

- Linux file system structure
- Basic terminal navigation
- File and directory management
- User and group permissions
- chmod and chown
- Running processes
- Package management
- Important directories like /etc, /var, /home
- Viewing logs and system information

---

## Tools Used

- Linux Terminal
- Bash Shell
- Command-line utilities

---

## Key Learnings

I learned how to navigate the Linux environment using terminal commands, manage files and directories, understand file permissions, and monitor running processes.

I also understood how Linux is widely used in servers, SOC environments, penetration testing, and security operations.
