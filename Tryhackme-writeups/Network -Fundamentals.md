# Room Name: Network Fundamentals

## Objective

The objective of this room was to understand the basic concepts of computer networking, including how devices communicate, how data travels across networks, and the importance of protocols, ports, and IP addressing in cybersecurity.

## Topics Covered

- What is a Network
- Private and Public IP Addresses
- IPv4 and IPv6 Basics
- MAC Address
- TCP and UDP Protocols
- Common Network Ports
- DNS (Domain Name System)
- HTTP and HTTPS
- Ping and Traceroute

## Tools Used

- Command Prompt
- Ping
- Traceroute
- Basic Networking Labs

## Key Learnings

I learned how systems communicate using IP addresses, ports, and protocols. I understood the difference between TCP and UDP, how DNS works, and why open ports are important in penetration testing.

## Practical Understanding

This room helped me understand the networking foundation required for cybersecurity, especially for reconnaissance, scanning, and vulnerability assessment.

This knowledge is also useful for my SentinelAI project in detecting open ports and analyzing security risks.
