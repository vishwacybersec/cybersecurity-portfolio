# SOC Investigation Queries

This document contains the Splunk Search Processing Language (SPL) queries used during the DNS log investigation.

---

## 1. Display All Events

```spl
index=dns_project
```

**Purpose:**
Displays all DNS events stored in the Splunk index.

---

## 2. Display Important DNS Fields

```spl
index=dns_project
| table timestamp src_ip dest_ip protocol port query query_type response_code
```

**Purpose:**
Displays only the important fields required for DNS investigation.

---

## 3. Count Total Events

```spl
index=dns_project
| stats count
```

**Purpose:**
Counts the total number of DNS events.

Result:

- Total Events = 12

---

## 4. Analyze Source IP Activity

```spl
index=dns_project
| stats count by src_ip
| sort -count
```

**Purpose:**
Shows how many DNS requests each source IP generated.

---

## 5. Analyze DNS Response Codes

```spl
index=dns_project
| stats count by response_code
```

**Purpose:**
Counts successful and failed DNS lookups.

Result:

- NOERROR = 8
- NXDOMAIN = 4

---

## 6. Investigate Failed DNS Requests

```spl
index=dns_project
| search response_code="NXDOMAIN"
```

**Purpose:**
Displays only failed DNS requests.

SOC analysts investigate these requests because repeated failures may indicate malware, phishing, or domain generation algorithms (DGA).

---

## 7. Identify Top Queried Domains

```spl
index=dns_project
| top query
```

**Purpose:**
Lists the most frequently queried domains.

---

## 8. Detect Repeated Domain Queries

```spl
index=dns_project
| stats count by query
| where count > 1
```

**Purpose:**
Identifies domains queried multiple times.

Repeated queries can indicate automated activity or malware.

---

## 9. Investigate Source IP

```spl
index=dns_project
| search src_ip="192.168.1.30"
```

**Purpose:**
Investigates all DNS activity generated by the selected source IP.

---

## 10. Investigate Suspicious Domain

```spl
index=dns_project
| search query="abcxyz123fake.com"
```

**Purpose:**
Displays all events involving the suspicious domain.

---

## 11. Analyze DNS Query Types

```spl
index=dns_project
| stats count by query_type
```

**Purpose:**
Shows the distribution of DNS query types.

Observed query types:

- A
- AAAA
- MX
- PTR
