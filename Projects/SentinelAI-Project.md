# SentinelAI – AI-Assisted Web Application Pentesting & Risk Analysis Platform

## Project Overview

SentinelAI is a cybersecurity project designed to perform ethical web application pentesting on authorized lab and test URLs only. It helps identify common security vulnerabilities and provides AI-assisted risk analysis with remediation suggestions.

The platform is built for educational, professional reporting, and entry-level security assessment purposes.

---

## Main Features

- Web vulnerability scanning
- SQL Injection detection
- Missing Security Headers detection
- Open Ports detection
- Risk severity classification
- AI-generated security analysis
- Remediation suggestions
- Professional PDF report generation
- Scan history with detail view
- Safe URL restriction logic

## Project Workflow

1. User enters a test URL
2. System validates safe testing scope
3. Vulnerability scanning begins
4. Findings are analyzed
5. AI generates risk summary
6. Professional report is generated
7. Results are stored in history
